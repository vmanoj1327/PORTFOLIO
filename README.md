# Manoj V — Portfolio

Personal portfolio website for **Manoj V**, B.Tech Computer Science student at Takshashila University. Built with HTML, CSS, and JavaScript.

## Live Demo

After enabling GitHub Pages, the site will be available at:

**https://vmanoj1327.github.io/PORTFOLIO/**

## Features

- Responsive dark/light theme
- Interactive 3D particle background and hero effects
- Sections: About, Vision, Skills, Education, Projects, Certifications, Experience, Achievements, Contact
- Smooth scroll animations and glassmorphism UI

## Tech Stack

- HTML5
- CSS3 (custom design system, animations)
- JavaScript (ES6+)
- [Three.js](https://threejs.org/) (CDN)
- [Font Awesome](https://fontawesome.com/) icons
- Google Fonts (Plus Jakarta Sans, Space Grotesk)

## Local Preview

1. Clone this repository
2. Open `index.html` in your browser, or use a local server:

```bash
# Python
python -m http.server 8000

# Node (npx)
npx serve .
```

3. Visit `http://localhost:8000`

## Deploy to GitHub Pages

1. Push this repo to `https://github.com/vmanoj1327/PORTFOLIO.git`
2. On GitHub: **Settings → Pages**
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**
4. Choose branch `main` and folder `/ (root)`
5. Save — your site will publish in a few minutes
